Cafe setup

Prerequisites

  1. NodeJs 0.6+ or later
  2. npm 1.2.14 or later
  3. Don't forget to add path to global node_modules folder in NODE_PATH. 
exports = NODE_PATH=$NODE_PATH:/usr/local/lib/node_modules
  1. Cafe interact's with npm api to download, install, wrap and bundle modules directly from the npm. So you may ensure than your current user has write permissions for folders ~/.npm and ~/tmp
    chown -R \`id -u\`:\`id -g\` ~/.npm ~/tmp
